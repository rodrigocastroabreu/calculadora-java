# 🧮 Calculadora Java

Aplicação de **calculadora com interface gráfica** desenvolvida em **Java**, utilizando **Swing** e conceitos de **Programação Orientada a Objetos (POO)**.

Este projeto foi desenvolvido como prática de programação em Java e também como parte do meu **portfólio de desenvolvimento no GitHub**.

---

# 📷 Interface

A aplicação apresenta uma **interface gráfica simples e intuitiva**, semelhante às calculadoras tradicionais de desktop.

Principais operações disponíveis:

* ➕ Soma
* ➖ Subtração
* ✖ Multiplicação
* ➗ Divisão
* 🔢 Entrada de números através de botões
* 🧹 Limpar visor

---

# 🛠 Tecnologias Utilizadas

* **Java**
* **Swing (Interface gráfica)**
* **Programação Orientada a Objetos (POO)**
* **Eclipse IDE**

---

# 📂 Estrutura do Projeto

O projeto foi organizado utilizando classes separadas para manter o código mais organizado e facilitar manutenção.

src/

Calculadora.java
Responsável pela lógica da calculadora e criação da interface.

Outras classes
Responsáveis por organizar eventos, operações matemáticas e execução do programa.

---

# 🚀 Como Executar

1. Clone este repositório:

git clone https://github.com/rodrigocastroabreu/calculadora-java.git

2. Abra o projeto em uma IDE Java (Eclipse, IntelliJ ou VS Code).

3. Execute a classe principal do projeto.

A interface gráfica da calculadora será aberta.
